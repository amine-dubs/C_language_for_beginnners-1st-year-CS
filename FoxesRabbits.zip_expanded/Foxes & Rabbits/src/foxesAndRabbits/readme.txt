Une simulation proie-prédateur mettant en jeu des renards (foxes) et des lapins (rabbits) 
sur un terrain rectangulaire fermé.

C'est la version de base de la simulation. Elle est écrite
sans utiliser l'héritage.

Comment démarrer :
    Créez un objet Simulator.
    Puis appelez l'une des méthodes :
        + simulateOneStep - pour un pas de simulation.
        + simulate - et donnez le nombre de semaines (disons 10) comme nombre de pas.
        + runLongSimulation - pour une simulation de 500 pas.
